# 🌾 CropClaim-AI

### Instant Crop Damage Assessment for PMFBY (Pradhan Mantri Fasal Bima Yojana)

---

## 📌 About the Project

**CropClaim-AI** is a cross-platform mobile application built with Flutter that helps farmers instantly assess crop damage and file insurance claims under the **PMFBY (Pradhan Mantri Fasal Bima Yojana)** scheme. The app leverages AI to simplify and speed up the crop damage assessment process, reducing delays in insurance settlements for farmers.

---

## 🚀 Features

- 📸 **AI-Powered Crop Damage Detection** — Upload or capture images of damaged crops for instant analysis
- 📋 **PMFBY Claim Assistance** — Guided step-by-step process to file crop insurance claims
- 🌍 **Cross-Platform** — Works on Android, iOS, Web, Windows, Linux, and macOS
- ⚡ **Instant Assessment** — Get quick damage reports without waiting for manual inspection
- 🧑‍🌾 **Farmer Friendly UI** — Simple and intuitive interface designed for ease of use

## 📦 Getting Started

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) installed
- Android Studio / VS Code with Flutter plugin
- A connected device or emulator

### Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/CropClaim-AI.git

# Navigate into the project
cd CropClaim-AI

# Get dependencies
flutter pub get

# Run the app
flutter run
```
